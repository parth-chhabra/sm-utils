LRU

A Simple LRU Map
This maintains 2 maps internally and swaps them when one becomes full
NOTE: At any time size of the map will be from 0 to 2 * maxItems

Example
const lru = new LRU({maxItems: 1000});
lru.set('hello', 'world');
lru.get('hello');
lru.delete('hello');
